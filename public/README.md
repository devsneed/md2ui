# md2ui

一个轻量级的文档渲染系统，将本地 Markdown 文档转换为美观的 HTML 页面。

## 项目简介

md2ui 是一个基于 Vue 3 开发的前端文档展示工具，专注于将 Markdown 文档以清晰、易读的方式呈现。它会自动扫描文档目录，构建导航树，并提供流畅的阅读体验。

## 核心特性

- **自动化目录** - 扫描文档文件夹，自动生成多级导航树
- **三栏布局** - 左侧导航、中间内容、右侧大纲，布局清晰
- **Markdown 增强** - 支持 GFM 语法、代码高亮、Mermaid 图表
- **交互优化** - 可拖拽调整宽度、折叠侧边栏、阅读进度显示
- **响应式设计** - 适配不同屏幕尺寸

## 技术栈

- Vue 3 + Vite
- Marked (Markdown 解析)
- Mermaid (图表渲染)
- Lucide Vue (图标库)

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

## 使用说明

### 文档组织

将你的 Markdown 文档放在 `public/docs/` 目录下：

```
public/docs/
├── 00-项目概览.md
├── 代码结构/
│   ├── 业务层/
│   └── 基础设施/
└── 架构设计/
    └── 01-核心流程.md
```

### 文件命名

使用 `序号-名称.md` 格式控制排序，例如：
- `00-项目概览.md`
- `01-快速开始.md`
- `02-架构设计.md`

序号越小，在导航中的位置越靠前。

### 首页内容

编辑 `public/README.md` 文件即可自定义首页内容。

## 功能说明

### 左侧导航
- 展示文档目录树结构
- 支持文件夹展开/折叠
- 可拖拽调整宽度
- 一键全部展开/收起

### 中间内容区
- 渲染 Markdown 文档
- 支持代码语法高亮
- 支持 Mermaid 流程图、时序图等
- 显示阅读进度和返回顶部按钮

### 右侧大纲
- 自动提取文档标题生成目录
- 点击跳转到对应章节
- 高亮当前阅读位置
- 可拖拽调整宽度

## 部署

构建后将 `dist` 目录部署到任何静态服务器：

```bash
pnpm build
```

支持部署到 Nginx、Apache、GitHub Pages、Vercel 等平台。
