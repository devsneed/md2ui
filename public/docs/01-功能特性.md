# 功能特性

md2ui 提供了丰富的功能，让文档阅读体验更加流畅。

## 三栏布局

界面采用经典的三栏布局设计：

- 左侧：文档导航树，支持多级目录
- 中间：文档内容区，渲染 Markdown
- 右侧：文档大纲，快速跳转章节

所有栏目都支持拖拽调整宽度，也可以折叠隐藏。

## Markdown 增强

支持 GitHub Flavored Markdown (GFM) 语法：

### 代码高亮

```javascript
// 支持多种语言的语法高亮
function greet(name) {
  return `Hello, ${name}!`
}
```

```python
# Python 代码示例
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```
