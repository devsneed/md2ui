# 自定义配置

md2ui 支持多种自定义配置，满足不同场景需求。

## 修改网站标题

编辑 `index.html`：

```html
<title>你的文档站点</title>
```

## 修改 Logo

编辑 `src/components/Logo.vue`：

```vue
<span class="logo-text">你的项目名</span>
```

或替换 `public/logo.svg` 图标文件。

## 修改首页

编辑 `public/README.md` 文件，这是点击 Logo 后显示的首页内容。

## 修改端口

编辑 `vite.config.js`：

```javascript
export default defineConfig({
  server: {
    port: 3000
  }
})
```

## 样式定制

全局样式在 `src/style.css` 中，主要变量：

```css
/* 颜色 */
--primary-color: #3b82f6;
--text-color: #1f2937;
--bg-color: #ffffff;
--border-color: #e5e7eb;

/* 尺寸 */
--sidebar-width: 280px;
--toc-width: 240px;
```

## 文档排序规则

文件名格式：`序号-名称.md`

- `00-快速开始.md` 排在最前
- `01-使用指南.md` 排在其后
- 没有序号的文件按字母排序

文件夹同样支持序号前缀。
