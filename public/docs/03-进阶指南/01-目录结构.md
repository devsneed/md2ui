# 目录结构

了解 md2ui 的项目结构，有助于你进行定制开发。

## 项目结构

```
md2ui/
├── public/                 # 静态资源
│   ├── README.md          # 首页内容
│   ├── logo.svg           # 网站图标
│   └── docs/              # 文档目录
├── src/
│   ├── api/               # API 接口
│   │   └── docs.js        # 文档列表获取
│   ├── components/        # Vue 组件
│   │   ├── Logo.vue       # Logo 组件
│   │   ├── TreeNode.vue   # 树形节点
│   │   ├── TableOfContents.vue  # 目录组件
│   │   └── ImageZoom.vue  # 图片放大
│   ├── composables/       # 组合式函数
│   │   ├── useMarkdown.js # Markdown 渲染
│   │   ├── useScroll.js   # 滚动处理
│   │   └── useResize.js   # 拖拽调整
│   ├── App.vue            # 主组件
│   ├── main.js            # 入口文件
│   └── style.css          # 全局样式
├── index.html             # HTML 模板
├── vite.config.js         # Vite 配置
└── package.json           # 项目配置
```

## 核心模块说明

### api/docs.js

负责扫描 `public/docs/` 目录，构建文档树结构。

### composables/useMarkdown.js

封装 Markdown 解析逻辑，包括：
- 使用 marked 解析 Markdown
- 提取标题生成目录
- 处理 Mermaid 代码块

### composables/useScroll.js

处理滚动相关逻辑：
- 计算阅读进度
- 高亮当前章节
- 返回顶部功能

### composables/useResize.js

实现拖拽调整宽度功能。
