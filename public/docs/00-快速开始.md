# 快速开始

欢迎使用 md2ui，本指南将帮助你在几分钟内启动项目。

## 环境要求

- Node.js 18+
- pnpm 8+

## 安装步骤

```bash
# 克隆项目
git clone https://github.com/your-username/md2ui.git
cd md2ui

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

启动后访问 http://localhost:5173 即可看到文档界面。

## 添加你的文档

将 Markdown 文件放入 `public/docs/` 目录：

```
public/docs/
├── 00-快速开始.md
├── 01-使用指南.md
└── 02-进阶配置/
    ├── 01-主题定制.md
    └── 02-部署方案.md
```

文件名前的数字用于控制排序，数字越小越靠前。

## 构建部署

```bash
# 构建生产版本
pnpm build

# 本地预览
pnpm preview
```

构建产物在 `dist/` 目录，可部署到任意静态服务器。
